# Sistema CRUD en Python

Proyecto básico de consola en Python que implementa un sistema CRUD
(Crear, Leer, Actualizar y Eliminar registros).

## Funcionalidades
- Crear registros
- Listar registros
- Actualizar registros
- Eliminar registros
- Menú interactivo en consola

## Tecnologías
- Python 3
- Programación modular

## Cómo ejecutar
```bash
python3 main.py
